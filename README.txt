How to run locally:

On your local machine run:

Install NodeJS -> http://nodejs.com
gem install compass
npm install -g bower (you may need to sudo)
npm install -g grunt-cli (you may need to sudo)

From the highwire directory run:

npm install (you may need to sudo)
bower install
grunt serve
